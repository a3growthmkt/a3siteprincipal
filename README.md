# A3 Growth - Assessoria de Marketing e Vendas

Landing page moderna e otimizada para conversão da **A3 Growth**, focada em escalar operações comerciais através de dados e processos previsíveis.

## 🚀 Tecnologias

- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript.
- **Backend**: Node.js com Express.js (preparado para deploy na Hostinger).
- **Design**: Premium Dark Gray Aesthetic, Grid 3x3 de Soluções, Metodologia de 3 etapas.

## 📦 Como rodar localmente

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório.
3. No terminal, execute:
   ```bash
   npm install
   npm start
   ```
4. O site estará disponível em `http://localhost:3000`.

## 🌐 Deploy (Hostinger)

Este projeto foi preparado para o modo **Node.js Web App** da Hostinger:
- O arquivo `server.js` gerencia as rotas e serve os arquivos estáticos.
- A porta é configurada automaticamente via variável de ambiente `PORT`.
- O script `npm start` é o ponto de entrada oficial.

## 🛠️ Estrutura de Soluções (A3 Framework)

O site apresenta o framework de 9 soluções da A3:
- A3 Traffic, A3 Page, A3 Creative, A3 Social, A3 Chat, A3 Voice (Em breve), A3 Insights, A3 Playbook, A3 CRM.
